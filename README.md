# Portafolio de Análisis de Datos - Carolina Rodríguez Guerra

¡Hola! Soy Carolina, analista de datos. Este portafolio muestra mi journey de aprendizaje y los proyectos que he desarrollado durante mi bootcamp en Tripleten, así como trabajos independientes. Los proyectos están organizados por **tipo de enfoque analítico** para demostrar diferentes habilidades: análisis completos (end‑to‑end), consultas SQL avanzadas y experimentos A/B.

---

## 📂 Estructura del portafolio

- end-to-end/ → Proyectos completos: limpieza, EDA, visualización y conclusiones.
- sql/ → Proyectos centrados en consultas SQL, optimización y bases de datos.
- ab-test/ → Experimentos controlados, pruebas de hipótesis y diseño estadístico.

## 📦 Configuración local

Para ejecutar cualquiera de los proyectos en tu máquina:

```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/data-analyst-portfolio.git

# Entrar al directorio
cd data-analyst-portfolio

# Instalar dependencias comunes
pip install -r requirements.txt

# Ejecutar Jupyter (para los proyectos end‑to‑end y A/B test)
jupyter notebook
```
